About Us

We are a group of college students enthusiastic in creating quality free tools and content on the Internet. The main purpose of this website is to provide the calories and nutrition values in any food item.We have used a nutrition api for all the nutrition value for any food .





In this project we have used python and django framework.

You have to type the name of any food item or mocktails in the search bar and click on the find calories button, and then you can check all its nutrition value and calories .
